# fresh ♻️
This installs everything Jonas needs after a fresh install of macOS!

# What does it install?
* Homebrew
* iTerm2
* Fish shell
* Spaceship fish theme
* Powerline Fonts
* Git
* IntelliJ
* CLion
* Spotify
* Notion
* Brave Browser
* Nextcloud Client
* Rust

# How to install

```
$ bash <(curl -s https://raw.githubusercontent.com/kegesch/fresh/master/install.sh)
```
